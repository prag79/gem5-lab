# gem5 Lab

Three guided exercises in computer-architecture simulation with
[gem5](https://www.gem5.org/), running entirely in your browser via
GitHub Codespaces. No local install required.

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/prag79/gem5-lab?quickstart=1)

## Students: start here

**[→ Read TUTORIAL.md](TUTORIAL.md)** — step-by-step walkthrough
of all three labs.

## Quick reference

Once your codespace is open:

    lab list    # show available labs
    lab 01      # SE-mode hello — build + run a static RV64 binary
    lab 02      # FS-mode Linux on a simulated HiFive board
    lab 03      # ATOMIC vs O3 CPU comparison on the same binary
    lab shell   # drop into a bash shell

Per-lab READMEs live under [`labs/`](labs/).
